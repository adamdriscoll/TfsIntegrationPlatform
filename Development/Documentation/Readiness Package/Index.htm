The configuration file templates are used in conjunction with the TFSMigrationShell and serve as excellent templates for the analysis and definition of new migration configurations.

 

The WSS and Custom POC Adapters are two custom sample adapters that serve as valuable examples for custom adapter developers, as possible starting points for new TFS Integration Tools adapters and as a guide through the configuration of a TFS Integration Platform development environment.
